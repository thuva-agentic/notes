Based on the files I found, this is a **local-first desktop notes application** called "Notes". Here's what it does:

## Overview

**Notes** is a privacy-focused, offline desktop note-taking app built with **Electron**, **React**, and **Tiptap**. All notes are stored locally on the user's machine — no account or cloud sync required.

## Key Features

- **Rich Markdown Editing**: Powered by Tiptap editor for comfortable markdown writing
- **Local Storage**: Notes saved as `.md` files in the OS app data directory (no cloud dependency)
- **Search**: Find notes quickly by searching across titles and content
- **Organization**: Nested folders for organizing notes
- **Sorting**: Sort by title (A-Z), last modified, or created date

## Storage Location

Notes are stored in the OS app data directory:
- **macOS**: `~/Library/Application Support/notes/notes/`
- **Windows**: `%APPDATA%/notes/notes/`
- **Linux**: `~/.config/notes/notes/`

Each note is a `.md` file with YAML frontmatter for title and dates.
