# Product Context

## Problem

Note-taking tools often require accounts, sync subscriptions, or store data in the cloud. Users who want a simple, private, offline notebook need a fast desktop app with solid markdown editing and organization.

## Solution

A desktop notes app that:

1. Stores all notes **locally** on the user's machine
2. Provides a **Tiptap**-powered editor for comfortable markdown writing
3. Lets users **search** across titles and content
4. Supports **sorting** (title, date created, date modified)
5. Supports **grouping/organization** (folders, tags, or notebooks — per Q&A)

## Users

- Individuals who want private, offline notes
- Developers and writers who prefer markdown
- Anyone avoiding cloud lock-in for personal notes

## UX principles

- **Fast capture** — new note in one action
- **Findable** — search is always accessible
- **Organized** — clear sidebar or list with sort/group controls
- **Native feel** — Electron desktop patterns (keyboard shortcuts, window chrome)
- **No sign-in friction** — open app and write
