# Notes — Project Brief

## Goal

Deliver a **local-first desktop notes app** for creating, editing, searching, sorting, and organizing markdown notes — no account or cloud required.

## Scope

- Cross-platform **Electron** desktop app
- Rich markdown editing via **Tiptap**
- Notes persisted **locally** on disk
- **Search**, **sort**, and **group/organize** (folders, tags, or collections — locked in ticket Q&A)
- Modern UI: **Tailwind CSS**, **shadcn/ui**, **Radix UI**, **React Icons**

## Out of scope (initial)

- Authentication / user accounts
- Cloud sync or multi-device sync
- Collaboration / shared notes
- Mobile-native apps

## Repository

- GitHub: `thuva-agentic/notes`
- Ticketing: GitHub Issues
- Base branch: `staging`

## Success criteria

Users can create and edit notes offline, find them quickly via search, and keep a personal library organized with sort and grouping controls.
